# q2ctf
This repo contains useful files for Quake 2 (Remaster) CTF (Capture the Flag).

## Getting started

### 1. Install Quake 2 from Steam

You need to install Quake 2. The best way is to install from ([Steam](https://store.steampowered.com/app/2320/Quake_II/)).
The game was remastered in 2023. When you install Steam version, you are automatically getting remastered version with CTF mode included.

### 2. Copy files

Next you need to, add couple useful files, which will make your game a little easier.
The most easy approach is to copy all files from ./baseq2 folder to your local baseq2 folder.

This is typical location on Windows machine

```
C:\Users\<user_name>\Saved Games\Nightdive Studios\Quake II\baseq2
```

## What are you getting

1. mac.cfg - my config file
2. server.cfg - config file to host CTF game
3. pak1.pak - custom pak with "full-bright" skins for CTF and weapons, for better visibility
3. ./maps - most popular CTF maps 
4. ./sound - timers for Quad Damage, Armor and Power Shield
